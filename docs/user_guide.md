# User Guide

#Login
1. Open the application.
2. Enter username and password.
3. Click Login.


#Managing Students
1. Open Student Form.
2. Add, update, or delete student records.
3. Save changes.


#Managing Courses
1. Open Course Form.
2. Enter course information.
3. Save changes.


#Managing Enrollments
1. Open Enrollment Form.
2. Select student and course.
3. Save enrollment.


#Reports
1. Open Report Form.
2. Select report type.
3. Generate report.
